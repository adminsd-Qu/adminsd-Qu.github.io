# LokiTank's Blog 🦎

个人博客，关注集成电路、微电子和嵌入式开发。

## 技术栈

- **框架**: [Next.js](https://nextjs.org/) 15 (App Router)
- **静态导出**: `output: 'export'` — 零服务端依赖
- **样式**: [Tailwind CSS](https://tailwindcss.com/) v4
- **字体**: Noto Sans SC + JetBrains Mono (通过 `next/font` 自托管)
- **内容**: Markdown + frontmatter
- **部署**: GitHub Actions → GitHub Pages

## 本地开发

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # 导出到 out/
```
